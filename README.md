## 🚀 项目简介

**Slavopolis Cloud** 是一个面向 Java 应用的现代化微服务脚手架项目，基于最新的 **JDK 21** + **Spring Boot 3.5.0** + **Spring Cloud 2025.0.0** 技术栈精心构建。项目涵盖多种业务组件深度封装、基础组件约定、第三方框架集成等，致力于帮助开发者快速搭建项目，将重心聚焦于业务实现。

### 🛠️ 安装步骤

<details>
<summary><b>📥 1. 克隆项目</b></summary>

```bash
git clone https://github.com/Slavopolis/slavopolis-cloud.git
cd slavopolis-cloud
```
</details>

<details>
<summary><b>🗄️ 2. 数据库初始化</b></summary>

```bash
# 创建数据库
mysql -u root -p
CREATE DATABASE slavopolis_cloud DEFAULT CHARSET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 导入初始化脚本
mysql -u root -p slavopolis_cloud < docs/sql/init.sql
```
</details>

<details>
<summary><b>⚙️ 3. 配置修改</b></summary>

```bash
# 修改配置文件
cp slavopolis-config/application-dev.yml.example slavopolis-config/application-dev.yml
# 根据实际环境修改数据库、Redis、RabbitMQ 等配置
```
</details>

<details>
<summary><b>🔨 4. 编译项目</b></summary>

```bash
mvn clean install -DskipTests
```
</details>

<details>
<summary><b>🚀 5. 启动服务</b></summary>

```bash
# 按顺序启动服务
# 1. 启动网关服务
cd slavopolis-gateway && mvn spring-boot:run

# 2. 启动认证服务
cd slavopolis-auth && mvn spring-boot:run

# 3. 启动系统服务
cd slavopolis-system && mvn spring-boot:run

# 4. 启动监控服务
cd slavopolis-monitor && mvn spring-boot:run
```
</details>

## 📖 开发指南

### 📏 代码规范

项目遵循 **阿里巴巴 Java 开发手册**，主要规范包括：

- 📝 包名使用小写字母，类名使用大驼峰命名
- 🔤 方法名使用小驼峰命名，常量使用大写字母加下划线
- 💬 注释完整，包括类注释、方法注释、参数说明
- ⚠️ 异常处理规范，统一错误码定义

### 🆕 新增服务

<details>
<summary><b>点击查看新增服务步骤</b></summary>

1. **创建服务模块**
```bash
mkdir slavopolis-your-service
cd slavopolis-your-service
```

2. **添加 pom.xml**
```xml
<parent>
    <groupId>com.slavopolis</groupId>
    <artifactId>slavopolis-cloud</artifactId>
    <version>1.0.0</version>
</parent>
<artifactId>slavopolis-your-service</artifactId>
```

3. **创建启动类**
```java
@SpringBootApplication
@EnableDiscoveryClient
public class YourServiceApplication {
    public static void main(String[] args) {
        SpringApplication.run(YourServiceApplication.class, args);
    }
}
```
</details>

### 📝 API 开发规范

- 使用 RESTful API 设计原则
- 统一返回格式 `Result<T>`
- 接口版本管理 `/api/v1/`
- 完整的参数校验
- Swagger 文档注解
